# Шахматное ателье

3D-ish шахматы для двух игроков с русским интерфейсом, локальным лидербордом и синхронизацией лидербордов между подключенными браузерами через WebSocket-сервер.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте `http://127.0.0.1:5174/`.

## Проверка

```bash
npm run build
npm run verify
```

## GitHub Pages

GitHub Pages запускает только статические файлы, поэтому там работают доска, локальная игра и сохраненный в браузере лидерборд. Мультиплеер между разными устройствами и автоматическая синхронизация лидербордов требуют Node/WebSocket-сервер (`npm run dev` или `npm run preview`) на отдельном хостинге.
